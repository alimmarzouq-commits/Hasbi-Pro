# حسابي Pro (Hissabi Pro) 💰

A professional financial tracker application built with React, Tailwind CSS, and Gemini AI.

## Features
- 📊 **Track Incomes and Expenses**: Easily add, categorize, and manage your daily transactions.
- 🤖 **AI Financial Advisor**: Powered by Gemini AI, get personalized financial insights and budgeting tips based on your spending habits.
- 🌙 **Dark/Light Mode**: Seamlessly switch between themes for comfortable viewing.
- 💱 **Multiple Currencies**: Choose your preferred currency (LYD, SAR, AED, EGP, USD, EUR).
- 📈 **Interactive Charts**: Visualize your financial flow with dynamic pie and bar charts.
- 💾 **Export to CSV**: Download your financial data for external accounting.

## Local Development

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Copy `.env.example` to `.env` and add your Gemini API key:
   ```bash
   GEMINI_API_KEY="your_api_key_here"
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion (Animations)
- Recharts (Data Visualization)
- Google Gen AI SDK (Gemini Integration)
